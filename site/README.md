# rustlab_llm — Lesson Site

A self-contained tutorial series for building a GPT-style language model
from first principles, using
[rustlab](https://github.com/kreegerresearch/rustlab) as the scripting
and visualisation environment. Each lesson is a single executable
Markdown notebook — derivations, code, and plots in one file.

What you see here is the **rendered** output of the source notebooks:
prose, executed `rustlab` blocks, captured `print()` output, and inline
SVG plots. GitHub displays it directly.

The source notebooks live at [`../notebooks/<slug>.md`](../notebooks/).
This `site/` tree is regenerated by `make notebooks` from the repo root;
only `site/README.md` (this file) is hand-written.

## Reading order

but not yet authored — see [`../PLAN.md`](../PLAN.md) for status.

## Local interactive view

For Plotly hover/zoom/pan and KaTeX-rendered math, run `make html` from
the repo root and open `index.html` (right next to this file) in a
browser. The HTML build includes an auto-generated entry page with
prev/next navigation between notebooks. The `index.html` and
per-notebook `*.html` files are gitignored — regenerate them locally
any time.

## How this is organized

The pattern (sources flat in `notebooks/`, rendered output committed to
a top-level `site/`, standalone `.r` scripts in `lessons/<slug>/`) is
documented at [`../rustlab/docs/lesson-site-pattern.md`](../../rustlab/docs/lesson-site-pattern.md)
and shared across the rustlab course family
([rustlab_em](../../rustlab_em/site/) uses the same layout).
